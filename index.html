Cours suivi

Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Internet, c’est le réseau. Le Web, c’est un des services qu’on utilise dessus, comme le mail ou les appels.

Internet

Internet est un réseau mondial d’ordinateurs interconnectés qui permet d’utiliser des services comme le web, les mails ou les appels.

Cloud

Le cloud permet d’utiliser des serveurs à distance pour stocker des données, faire tourner des applications ou accéder à des services.

W3C

World Wide Web Consortium

Partie 2 - Distinguez les langages du Web

Langages client

Ce sont des langages exécutés dans le navigateur de l'utilisateur, comme HTML, CSS ou JavaScript, pour afficher et animer les pages web.(Front-End)

Navigateurs

Un navigateur est un logiciel (comme Chrome, Firefox, Safari) qui permet d’afficher et d’interagir avec des sites web.

Langages serveur

Ce sont des langages exécutés sur un serveur, comme PHP, Python ou Node.js, qui génèrent les pages web avant qu’elles n’arrivent au navigateur.(Back-End)

Frameworks

Un framework est un ensemble d’outils et de règles qui aident à créer des applications plus rapidement, sans repartir de zéro.

Bases de données

Une base de données permet de stocker, organiser et retrouver des informations utilisées par un site ou une application (comme les comptes utilisateurs ou les produits).

Site responsive

Un site responsive est un site qui s’adapte automatiquement à tous les écrans : ordinateurs, tablettes, téléphones.

Application mobile

Une application mobile est un programme installé sur un smartphone ou une tablette, conçu pour fonctionner sur iOS (Apple) ou Android.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur puissant qui stocke des données et répond aux demandes des autres ordinateurs via Internet (comme envoyer une page web à un navigateur).

Adresse IP

Une adresse IP est un numéro unique qui identifie un appareil sur un réseau, un peu comme une adresse postale sur Internet.

Nom d’hôte et DNS

Le nom d’hôte est l’adresse lisible d’un site (ex. : google.com) et le DNS est le service qui le traduit en adresse IP pour que les navigateurs puissent le trouver.

Protocoles

Les protocoles sont des règles de communication utilisées sur Internet, comme HTTP (pour les pages web) ou FTP (pour transférer des fichiers).

Partie 4 - Identifiez les métiers du web

Développeurs

Les développeurs écrivent le code des sites ou applications, que ce soit pour la partie visible (frontend) ou la partie serveur (backend).

Product Managers et chefs de projets

Ils organisent et supervisent la création d’un site ou d’une app, en gérant l’équipe, les délais et les besoins des utilisateurs.

Designers

Les designers s’occupent de l’apparence visuelle d’un site ou d’une app, en rendant l’interface claire, agréable et intuitive.

Testeurs

Les testeurs vérifient que le site ou l’application fonctionne correctement en repérant les bugs et les problèmes d’affichage ou de navigation.

Techniciens informatique

Ils assurent le bon fonctionnement des réseaux, ordinateurs et serveurs, souvent en entreprise ou dans des équipes web.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

CSS est le langage qui permet de styliser les pages web, en définissant les couleurs, tailles, marges, polices, etc.

JavaScript

Front-End

JavaScript est un langage qui rend les pages web interactives et dynamiques (animations, clics, formulaires...).

JavaScript

Back-End

JavaScript peut aussi être utilisé côté serveur (avec Node.js) pour gérer les requêtes, les bases de données et créer des API.

PHP

Back-End

PHP est un langage serveur qui permet de générer des pages web dynamiques et de gérer des bases de données.

Java

Back-End

Java est un langage robuste et polyvalent utilisé pour développer des applications web, mobiles et d’entreprise côté serveur.

Python

Back-End

Python est un langage simple et puissant, utilisé côté serveur pour créer des sites, gérer des bases de données ou développer des scripts.

C#

Back-End

C# est un langage de Microsoft utilisé pour développer des applications web, desktop et back-end, souvent avec le framework .NET.

SQL

Bases de données

SQL est un langage qui permet de créer, modifier et interroger des bases de données pour gérer des informations.

Boîte à outils du développeur

  • IDE comme VS Code : Sublim Code, IntelliJ, PyCharm, Atom...
  • Github : GitHub est une plateforme en ligne qui permet de stocker, partager et collaborer sur du code grâce au système de version Git.
  • Librairies / bibliothèques : Une librairie est un ensemble de fonctions ou de morceaux de code prêts à l’emploi qu’on peut utiliser pour faciliter le développement.
  • Frameworks : Un framework est un cadre de travail avec des outils et des règles pour développer plus vite une application sans repartir de zéro.Exemples :Laravel (PHP);Django (Python); Angular (JavaScript).
